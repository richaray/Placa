# 🌿 Placa

Snap or upload a photo of any plant — Placa identifies the species and gives you tailored care advice in seconds, powered by Google's Gemini Vision model.

![Gemini Vision](https://img.shields.io/badge/Gemini-Vision%20API-4285F4?style=flat&logo=google)
![Netlify](https://img.shields.io/badge/Deployed-Netlify-00C7B7?style=flat&logo=netlify)

---

Website Link : https://placa-plants.netlify.app/

## ✨ Features

- 📸 **Image-based plant identification** — upload or snap directly from your device
- 💧 **Tailored care advice** — watering schedules, light requirements, soil type, and common mistakes to avoid
- 🌱 **Fun fact** per plant identification
- ⚡ **Fast responses** — results in seconds via Gemini Vision API
- ☁️ **Serverless backend** — Netlify function handles all API calls, keeping your key off the client

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Vanilla JS, HTML, CSS |
| Backend | Netlify Serverless Function (Node.js) |
| AI Model | Google Gemini Vision API (`gemini-1.5-flash` with fallback) |
| Deployment | Netlify |
